# 导出 App 全部文案

setp1: 安装 androguard
pip install -U androguard
step2：导出全部英语文案
androguard arsc [apk路径] --type string -o en-strings.xml
step3:在线转换为 csv https://www.aconvert.com/cn/document/xml-to-csv/
